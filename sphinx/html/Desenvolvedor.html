Guia Desenvolvedor¶

Para o desenvolvedor é recomendado a criação de um ambiente virtual python para encapsular melhor o ambiente de trabalho. Exemplo: <https://uoa-eresearch.github.io/eresearch-cookbook/recipe/2014/11/26/python-virtual-env/>

Dependências¶

$pip install sqlite3

$pip install flask

$pip install flask_httpauth

Para rodar¶

python softdes.py
